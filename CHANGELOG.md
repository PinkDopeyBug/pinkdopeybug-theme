## 中文版更新日志

### 更新日志 - PinkDopeyBug 主题插件 0.0.1 版本

**发布日期:** 2024年10月27日
**版本号:** 0.0.1
**作者:** PinkDopeyBug
**贡献者:** PinkDopeyBug

---

#### 更新内容

- **初始发布**
  - 基于 Atom One Dark 和 One Dark Modern Classic 主题创建了 PinkDopeyBug 主题。
  - 解决了 One Dark Modern Classic 在 XML 和 YML 文件中缩进线条缺失的问题。
  - 改进了光标悬停和文本选中的背景色对比度，以增强代码的突出度。
  - 调整了注释颜色以提高在深色背景下的可读性。
  - 为了减少从深色编码环境切换到其他浅色应用程序时的眼部不适，采用了 One Dark Modern Classic 的较浅背景色。
  - 调整了变量颜色，以避免与错误信息混淆。
  - 提供了详细的安装和使用说明。

## English Version Change Log

### Changelog - PinkDopeyBug Theme Plugin 0.0.1 Release

**Release Date:** October 27, 2024
**Version:** 0.0.1
**Author:** PinkDopeyBug
**Contributors:** PinkDopeyBug

---

#### What's New

- **Initial Release**
  - Created the PinkDopeyBug theme based on Atom One Dark and One Dark Modern Classic themes.
  - Fixed the issue of missing indentation lines in XML and YML files within One Dark Modern Classic.
  - Enhanced the contrast of cursor hover and text selection backgrounds to make code stand out more.
  - Adjusted comment colors to improve readability against a dark background.
  - Adopted a lighter background color from One Dark Modern Classic to reduce eye strain when switching between dark coding environments and light-colored applications.
  - Changed variable colors to avoid confusion with error messages.
  - Provided detailed installation and usage instructions.
